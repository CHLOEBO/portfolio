Black Friday Review Generator

Try my project: Black Friday Review.

This project takes reviews and evaluates whether they are mainly positive or negative. It also asigns a star rating based on the review. If you want, you can generate a fake reviews which replaces the adjectives with new ones.

What I learned during my project...

  • sentimentVal finds the corresponding decimal value for the word, which in my example is "terrible".
  • totalSentimentVal finds the sum of the sentiment value for each word in the list. It checks, using a for loop, where each word ends and then finds its sentiment value.
  • starRating takes the totalSentimentVal of a review and assigns it a rating on a scale of 1 to 4 based on the value of the review. A review with a higher value will get a higher rating
  • fakeReview works by finding each word which starts with an asterisk. It uses a for loop to find an asterisk. Once it does, it uses a nested for loop to see when the word ends. The word is then replaced by
